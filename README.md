# Personal Resume Website

This is a GitHub Pages-ready personal website template.

## Edit your resume content
Update `resume-content.js` with your real resume information.

## Publish to GitHub
```bash
git init
git add .
git commit -m "Initial personal website"
git branch -M main
git remote add origin https://github.com/<your-username>/<repo-name>.git
git push -u origin main
```

Then enable Pages in GitHub:
Settings -> Pages -> Deploy from a branch -> `main` and `/ (root)`.
