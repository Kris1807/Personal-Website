# Kristian Pitshugin - Personal Website

Personal portfolio/resume website built as a static site and hosted with GitHub Pages.

## Live Site
[https://kris1807.github.io/Personal-Website/](https://kris1807.github.io/Personal-Website/)

## Repository
[https://github.com/Kris1807/Personal-Website](https://github.com/Kris1807/Personal-Website)

## Tech
- HTML
- CSS
- JavaScript

## Project Structure
- `index.html` - Page layout and section rendering
- `styles.css` - Visual design and responsive layout
- `resume-content.js` - Resume/profile data used by the site

## Update Content
Edit `resume-content.js` to change:
- Contact info
- Experience
- Projects
- Education
- Skills

## Run Locally
Open `index.html` directly in your browser.

## Deploy (GitHub Pages)
1. Push changes to `main`
2. In GitHub, open `Settings -> Pages`
3. Set:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
4. Save and wait for deployment
